Key Elements

Health Indicators (in log header)
Green/yellow/red dots for WS, ZMQ, and Process status. Plus throughput and last heartbeat.
Event Log Stream
Color-coded by type: BAR, ORDER, FILL, SIGNAL, ERROR, HEARTBEAT. Auto-scrolls to latest.
Filter Bar
Toggle event types on/off. Heartbeats hidden by default to reduce noise.
Collapsible
Click the header to collapse/expand. Remembers state. Arrow indicator shows open/closed.
